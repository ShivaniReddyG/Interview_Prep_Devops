# Top 100 Amazon Web Services Interview Questions for DevOps & Cloud Engineers

# AWS Fundamentals

### 1. What is AWS?

AWS is a cloud computing platform that provides services like compute, storage, networking, databases, security, and DevOps tools.

---

### 2. What are the benefits of AWS?

* Scalability
* High availability
* Pay-as-you-go pricing
* Global infrastructure
* Security
* Automation

---

### 3. What is a Region in AWS?

A geographical area containing multiple Availability Zones.

---

### 4. What is an Availability Zone?

An isolated data center within a Region.

---

### 5. Difference between Region and Availability Zone?

| Region              | Availability Zone         |
| ------------------- | ------------------------- |
| Geographic location | Data center within region |
| Multiple AZs        | Physically isolated       |

---

### 6. What is an Edge Location?

AWS CDN locations used by CloudFront for content caching.

---

### 7. What is IAM?

Identity and Access Management service used to control AWS access securely.

---

### 8. Difference between IAM Role and IAM User?

| IAM User           | IAM Role                  |
| ------------------ | ------------------------- |
| Permanent identity | Temporary permissions     |
| Login credentials  | Assumed by services/users |

---

### 9. What is a Security Group?

Virtual firewall controlling inbound and outbound traffic.

---

### 10. Difference between Security Group and NACL?

| Security Group | NACL         |
| -------------- | ------------ |
| Instance level | Subnet level |
| Stateful       | Stateless    |

---

# EC2 (Elastic Compute Cloud)

### 11. What is EC2?

Virtual server service in AWS.

---

### 12. Types of EC2 instances?

* General Purpose
* Compute Optimized
* Memory Optimized
* Storage Optimized
* GPU Instances

---

### 13. What is AMI?

Amazon Machine Image used to launch EC2 instances.

---

### 14. Difference between EBS and Instance Store?

| EBS              | Instance Store      |
| ---------------- | ------------------- |
| Persistent       | Temporary           |
| Network attached | Physically attached |

---

### 15. What is Auto Scaling?

Automatically adjusts EC2 instances based on load.

---

### 16. What is an Elastic IP?

Static public IPv4 address.

---

### 17. What is Load Balancer?

Distributes traffic across multiple targets.

---

### 18. Types of Load Balancers?

* ALB
* NLB
* CLB
* GWLB

---

### 19. Difference between ALB and NLB?

| ALB        | NLB     |
| ---------- | ------- |
| Layer 7    | Layer 4 |
| HTTP/HTTPS | TCP/UDP |

---

### 20. How do you secure EC2 instances?

* Security Groups
* IAM roles
* Patching
* SSH key authentication
* Disable root login

---

### 21. What is S3?

Object storage service.

---

### 22. What are S3 storage classes?

* Standard
* IA
* One Zone IA
* Glacier
* Glacier Deep Archive

---

### 23. What is versioning in S3?

Maintains multiple versions of objects.

---

### 24. What is lifecycle policy in S3?

Automates object transition and deletion.

---

### 25. Difference between S3 and EBS?

| S3              | EBS             |
| --------------- | --------------- |
| Object storage  | Block storage   |
| Unlimited scale | Attached to EC2 |

---

### 26. What is multipart upload?

Large file upload in chunks.

---

### 27. What is S3 replication?

Automatic copying between buckets/regions.

---

### 28. What is a pre-signed URL?

Temporary secure access URL.

---

### 29. How do you secure S3 buckets?

* Bucket policies
* IAM policies
* Encryption
* Block public access

---

### 30. Difference between SSE-S3 and SSE-KMS?

| SSE-S3           | SSE-KMS          |
| ---------------- | ---------------- |
| AWS managed keys | KMS managed keys |

---

### 31. What is VPC?

---

### 32. What is a subnet?

---

### 33. Difference between public and private subnet?

| --------------- | ------------------ |
| Internet access | No direct internet |

---

### 34. What is an Internet Gateway?

Enables internet access for VPC.

---

### 35. What is NAT Gateway?

Allows private subnet outbound internet access.

---

### 36. What is route table?

Defines traffic routing rules.

---

### 37. What is VPC Peering?

Connects two VPCs privately.

---

### 38. Difference between Security Group and NACL?

Security Group is stateful; NACL is stateless.

---

### 39. What is Direct Connect?

Dedicated private AWS network connection.

---

### 40. What is Transit Gateway?

Central hub connecting multiple VPCs.
